## Implementation Techniques

### Reference: CuTe DSL (`conf/fixtures/matmul/cute_gemm.py`)

CUTLASS CuTe DSL with MLIR JIT compilation.

- **Tile**: 128×128×64 (M×N×K), 4 MMA warps + 1 DMA = 160 threads
- **Pipeline**: 3-stage PipelineTmaAsync with mbarrier
- **Data movement**: TMA G2S → ldmatrix S2R → mma.sync m16n8k16
- **Scheduling**: StaticPersistentTileScheduler (188 CTAs)
- **Key techniques**: warp specialization, BF16 epilogue, CuTe tensor caching
- **Registers**: 207/thread (NCU measured)

### Generated: Hand-Written CUDA PTX (`data/generated/sm120/matmul/generated.cu`)

Two size-specialized kernel variants with inline PTX.

- **Big** (M,N ≥ 2048): 256×128×32 tile, 256 threads (8 warps), 3 stages, 191 regs
- **Small** (M,N < 2048): 128×64×32 tile, 128 threads (4 warps), 4 stages, 134 regs
- **Data movement**: TMA via inline PTX + mbarrier + ldmatrix + mma.sync
- **Key techniques**: interleaved A-load/MMA, SMEM swizzle (64B/128B), 4×4 super-tile L2 ordering, multi-variant dispatch

### cuBLAS: Vendor Baseline (`conf/fixtures/matmul/cudnn.py`)

- `torch.mm()` → cuBLAS `cublasGemmEx`, BF16 with FP32 accumulation
- Closed-source, CUTLASS-based (256×128 tile, 32 K-tile, 3 stages)

## Key Findings

1. **Generated beats cuBLAS at ALL sizes** — from 2.25x at 256 to 1.00x at 8192
2. **Reference improved 1.01-2.24x** from CuTe tensor caching (eliminates ~17μs Python dispatch)
3. **Reference gap at large sizes (5-10%)** is structural: 128×128 tile / 10.4% occupancy vs 256×128 / 16.7%
4. **Larger tiles blocked for CuTe DSL** — 256×128 causes 255 regs + spill (NCU verified)
5. **TILE_K=64 blocked for generated** — mbarrier limited to 3 TMA loads/phase on SM120

## Hardware Constraints Discovered

- **mbarrier TMA limit**: max 3 `cp.async.bulk.tensor` completions per mbarrier phase (>3 → deadlock/crash)
- **CuTe DSL tile limit**: bM > 128 causes register spill to 255 with atom_layout (2,2,1)
- **SMEM capacity**: 100 KB per SM (configurable via cudaFuncSetAttribute)
- **TMA box dim0 limit**: 32 elements with 64B swizzle, 64 elements with 128B swizzle
