# NVIDIA B200 (HGX) — Roofline Parameters

Source: `docs/roofline/nvidia-b200-datasheet.pdf` (page 7, Individual Blackwell GPU Specifications)

Note: All Tensor Core numbers below include sparsity per NVIDIA's datasheet convention.
Dense values are estimated at half the sparse values.

| Parameter | Value |
|---|---|
| GPU | GB200 (Blackwell) |
| SMs | 160 |
| CUDA Cores | 20,480 |
| Tensor Cores | 640 (5th Gen) |
| TDP | 1,000 W (HGX config) |
